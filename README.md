## 🎯 אודות הפרויקט

EASYCAL היא אפליקציית הזמנת תורים למספרות גברים. האפליקציה מאפשרת ללקוחות לחפש מספרות, לצפות בפרטים ולהזמין תורים בקלות.

## 🚀 התחלה מהירה

### Vercel Deployment
1. העלה את התיקייה ל-GitHub
2. חבר את ה-Repository ל-Vercel
3. Vercel יזהה אוטומטית שזה אתר סטטי ויעלה אותו

### Local Development
פשוט פתח את `index.html` בדפדפן.

## 📁 מבנה הפרויקט

```
easycal/
├── index.html      # אפליקציית הלקוחות
├── barber.html     # אפליקציית הספרים
└── README.md       # קובץ זה
```

## ✨ פיצ'רים

### אפליקציית לקוחות (index.html)
- 🔍 חיפוש מספרות
- ❤️ שמירת מועדפים
- 📅 הזמנת תורים
- 👤 פרופיל משתמש

### אפליקציית ספרים (barber.html)
- 📊 דשבורד סטטיסטיקות
- 📅 יומן תורים
- 👤 ניהול פרופיל
- ✂️ ניהול שירותים
- ⏰ ניהול שעות פעילות

## 🛠️ טכנולוגיות

- React 18 (via CDN)
- Tailwind CSS (via CDN)
- Lucide Icons
